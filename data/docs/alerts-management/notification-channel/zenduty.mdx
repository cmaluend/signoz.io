## Integration with Zenduty

1. On Zenduty, go to Teams on Zenduty and click on the team you want to add the integration to
2. Next, go to Services and click on the relevant Service
3. Go to Integrations and then Add New Integration. Give it a name and select the application Prometheus from the dropdown menu.
4. Go to Configure under your Integrations and copy the Webhook URL generated
5. Copy the webhook URL and use it in the SigNoz Alertmanager webhook integration
6. On SigNoz, Click on the "Test" button to test the integration

## Configure programmatically

Since this integration uses the Webhook channel, you can configure it programmatically using the same mechanism as described in the [Webhook](/docs/alerts-management/notification-channel/webhook#configure-programmatically) section.

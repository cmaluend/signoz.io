This integration enables monitoring of your AWS DynamoDB tables by collecting comprehensive metrics. With SigNoz's one-click AWS integration, you can track capacity utilization, request latency, and throttling events.

Amazon DynamoDB is a fully managed NoSQL database service that provides fast and predictable performance with seamless scalability. Monitoring DynamoDB is critical for capacity planning, performance optimization, and cost management.

## Getting Started

To use the DynamoDB integration:

1. From the sidebar, navigate to **Integrations**.
2. Click **Configure** on the **Amazon Web Services** tile.
3. Navigate to **DynamoDB** to view dashboards and configure the integration.

For detailed setup instructions, see [One-Click AWS Integrations](https://signoz.io/docs/integrations/aws/one-click-aws-integrations/).

## What You'll Monitor & Data Collected

This integration collects comprehensive DynamoDB metrics, providing visibility into:

### Core DynamoDB Metrics
- **Capacity Utilization**: Monitor read and write capacity usage.
- **Request Latency**: Track successful request latency.
- **Throttling**: Monitor throttled requests and write throttle events.
- **Errors**: Track user errors and system errors.

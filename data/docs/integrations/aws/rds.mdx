This integration enables monitoring of your Amazon RDS database instances by collecting comprehensive database performance metrics. With SigNoz's one-click AWS integration, you can track CPU usage, connections, storage, and I/O performance.

Amazon RDS is a managed relational database service supporting multiple database engines. Monitoring RDS is essential for understanding database health, query performance, and resource utilization.

## Getting Started

To use the RDS integration:

1. From the sidebar, navigate to **Integrations**.
2. Click **Configure** on the **Amazon Web Services** tile.
3. Navigate to **RDS** to view dashboards and configure the integration.

For detailed setup instructions, see [One-Click AWS Integrations](https://signoz.io/docs/integrations/aws/one-click-aws-integrations/).

## What You'll Monitor & Data Collected

This integration collects comprehensive RDS metrics, providing visibility into:

### Core RDS Metrics
- **CPU Utilization**: Monitor CPU usage across database instances.
- **Database Connections**: Track current and max connection counts.
- **Storage**: Monitor free storage space and burst balance.
- **I/O Performance**: Track read/write IOPS and latency.
- **Memory**: Monitor freeable memory.
- **Network**: Track network throughput.

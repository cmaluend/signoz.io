This integration enables monitoring of your AWS Lambda functions by collecting comprehensive serverless compute metrics. With SigNoz's one-click AWS integration, you can track invocations, errors, duration, and concurrency.

AWS Lambda is a serverless compute service that runs your code in response to events. Monitoring Lambda is essential for understanding function performance, identifying errors, and optimizing costs.

## Getting Started

To use the Lambda integration:

1. From the sidebar, navigate to **Integrations**.
2. Click **Configure** on the **Amazon Web Services** tile.
3. Navigate to **Lambda** to view dashboards and configure the integration.

For detailed setup instructions, see [One-Click AWS Integrations](https://signoz.io/docs/integrations/aws/one-click-aws-integrations/).

## What You'll Monitor & Data Collected

This integration collects comprehensive Lambda metrics, providing visibility into:

### Core Lambda Metrics
- **Invocations**: Monitor function invocation counts.
- **Errors**: Track error counts and rates.
- **Duration**: Monitor execution time.
- **Throttles**: Track throttled invocations.
- **Concurrency**: Monitor concurrent executions.
- **Async Events**: Track async event processing.

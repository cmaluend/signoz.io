This dashboard provides monitoring of Envoy Proxy performance using <a href="https://www.envoyproxy.io/docs/envoy/latest/configuration/observability/stat_sinks/open_telemetry_stat_sink" target="_blank" rel="noopener noreferrer nofollow">OpenTelemetry Stat Sink</a>, giving you visibility into traffic patterns, connection handling, and internal server health.

<Admonition type="info">
To use this dashboard, you need to configure Envoy to send metrics to SigNoz. Follow the [Envoy Metrics](https://signoz.io/docs/userguide/envoy-metrics/) guide to get started.
</Admonition>

## Dashboard Variables

This dashboard includes pre-configured variables for filtering:

- **namespace**: Filter by Kubernetes namespace
- **service.name**: Filter by specific service name
- **cluster**: Filter by upstream cluster name
- **deployment_environment**: Filter by deployment environment (e.g., production, staging)

## Metrics Included

### General Overview

- **Active Connections**: Active downstream HTTP connections
- **Total Requests**: Total HTTP requests received
- **Request Rate**: HTTP requests per second
- **Envoy Uptime**: Envoy process uptime

### Traffic & Latency

- **Ingress Data Rate**: Incoming data rate (bytes/s)
- **Egress Data Rate**: Outgoing data rate (bytes/s)
- **Average Response Size**: Average HTTP response size
- **Request Throughput**: Average request throughput
- **Average Response Time**: Mean HTTP request latency
- **P50 Latency**: 50th percentile latency
- **P95 Latency**: 95th percentile latency
- **P99 Latency**: 99th percentile latency

### Errors & Responses

- **Response Code Distribution**: Response codes by class
- **2xx Success Rate**: Rate of successful responses
- **4xx Client Errors**: Rate of client errors
- **5xx Server Errors**: Rate of server errors
- **Total Error Count**: Cumulative 4xx and 5xx count
- **Error Rate**: Combined 4xx and 5xx error rate
- **Error Rate by Class**: Breakdown of 4xx vs 5xx errors

### Upstream Metrics

- **Upstream Request Volume**: Requests by upstream cluster
- **Upstream Active Connections**: Active upstream connections
- **Upstream Connection Failures**: Failed upstream connections
- **Upstream Error Rate**: Upstream 4xx/5xx errors
- **Upstream Throughput**: Total TX+RX to upstream clusters

### Connection Lifecycle

- **Connection Establish Rate**: New downstream connections/sec
- **Connection Destruction Rate**: Closed downstream connections/sec

### Resource Usage

- **Memory Allocated**: Envoy memory usage (allocated)
- **Heap Size**: Envoy heap memory size

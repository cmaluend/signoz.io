## What This Dashboard Monitors

This dashboard tracks essential Kubernetes pod metrics to help you:

- **Monitor Pod Performance**: Track CPU and memory usage for individual pods
- **Resource Optimization**: Analyze memory consumption patterns across pods
- **Filesystem Management**: Monitor filesystem usage, available space, and capacity

## Metrics Included

### CPU Metrics
- **Pod CPU Usage**: Real-time CPU utilization for each pod

### Memory Metrics
- **Pod Memory Working Set**: Active memory currently being used by pod processes
- **Pod Memory RSS**: Resident Set Size memory usage per pod
- **Pod Memory Usage**: Current memory consumption per pod

### Filesystem Metrics
- **Pod Filesystem Usage**: Current storage consumption by pod containers
- **Pod Filesystem Available**: Available storage space for each pod
- **Pod Filesystem Capacity**: Total filesystem capacity allocated to pods

## Dashboard Variables

This dashboard includes pre-configured variables for filtering:

- **k8s.cluster.name**: Filter by Kubernetes cluster name
- **k8s.node.name**: Filter by specific node names
- **k8s.namespace.name**: Filter by specific namespace
